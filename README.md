## ✨ What is PyMarket?

**PyMarket** is a fully-featured Python project marketplace where developers can browse, purchase, and download ready-made Python projects. Each project is sold exclusively to **one buyer only** — you get full source code, lifetime access, and real ownership.

Payments are handled via **GCash** 💙, making it seamless for Filipino buyers. The platform also features a **live auction system** where users can bid on exclusive projects in real time.

## 🤖 PyMarket Bot

The built-in support bot automatically handles common questions when the admin is away:

<div align="center">

| Trigger Keywords | Bot Response |
|---|---|
| `hi`, `hello`, `hey`, `kumusta` | Time-aware greeting + intro ☀️🌙 |
| `price`, `project`, `magkano` | Lists all available projects with prices |
| `gcash`, `pay`, `bayad` | Step-by-step GCash payment guide 💳 |
| `order`, `status`, `pending` | Real-time order status check 📦 |
| `auction`, `bid`, `win` | Full auction guide 🏷️ |
| `download`, `file`, `source` | How to access purchased projects 📥 |
| `thank`, `salamat` | Warm thank-you reply 🙏 |
| *(anything else)* | Friendly fallback with topic menu 🚀 |

</div>

## 📁 Project Structure

```
pymarket/
├── 📄 app.py                   # Main Flask application
├── 🤖 bot.py                   # Auto-reply support bot
├── 📄 wsgi.py                  # PythonAnywhere deployment config
├── 📋 requirements.txt         # Dependencies (flask, werkzeug)
├── 🗄️  market.db                # SQLite database (auto-created)
│
├── 📂 static/uploads/
│   ├── screenshots/            # Project preview images
│   ├── projects/               # Downloadable project files
│   ├── payments/               # GCash payment screenshots
│   └── avatars/                # User profile photos
│
└── 📂 templates/
    ├── base.html               # Shared layout, navbar, dark/light mode
    ├── index.html              # Browse page
    ├── project_detail.html     # Project detail + buy button
    ├── buy.html                # GCash payment page
    ├── auctions.html           # Auction listings
    ├── auction_detail.html     # Live bidding page
    ├── my_orders.html          # Order history
    ├── chat.html               # Support chat + bot messages
    ├── profile.html            # User profile
    └── admin/                  # Admin panel templates
```
